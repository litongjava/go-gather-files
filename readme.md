# go-gather-files

将当前目录下及其目录子的所文件都收集到一个文件中
存储格式是

```
文件名
----------------------------
文件内容
----------------------------

子录名/文件名
----------------------------
文件内容
----------------------------

子目录名/子子目录名/文件名
-------------------------------
文件内容
-------------------------------
```

## usage

```
go-gather-files -I node_modules,.idea,public -o output_1.txt E:\code\project\esunward\gaozhuang-html\sanban-real-world
```
